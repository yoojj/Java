# JPA
Java Persistence API   
: ORM(Object-Relational Mapping)을 위한 스펙 제공       
: 구현체에 따라 NoSQL DB 일부 지원       
: EJB 스펙 일부였으나 Java EE 6부터 분리되어 별도 사양으로 존재           


**구현체**  
- Hibernate
- EclipseLink
- OpenJPA
- ...
